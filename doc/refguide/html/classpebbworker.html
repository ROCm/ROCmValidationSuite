Detailed Description

Bandwidth test implementation class.

Derives from rvs::ThreadBase and implements actual test functionality in its run() method.

int pebbworker::do_transfer ( )

Executes data transfer.

Based on transfer parameters, initiates and performs one way or bidirectional data transfer. Resulting measurements are compounded in running totals for periodical printout during the test.

Returns
0 - if successfull, non-zero otherwise
