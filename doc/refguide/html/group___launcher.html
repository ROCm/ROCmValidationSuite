Detailed Description

Launcher module implementation.

This is the starting point of rvs utility. Launcher will parse the command line as well as the input YAML configuration file and load appropriate test modules. Then, it will invoke tests specified in .conf file and provide logging functionality so that modules can print on screen of in the log file.
