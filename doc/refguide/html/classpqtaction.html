Detailed Description

PQT action implementation class.

Derives from rvs::actionbase and implements actual action functionality in its run() method.

int pqtaction::create_threads ( )
protected

Create thread objects based on action description in configuation file.

Threads are created but are not started. Execution, one by one of parallel, depends on "parallel" key in configuration file. Pointers to created objects are stored in "test_array" member

Returns
0 - if successfull, non-zero otherwise
